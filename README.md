#### 为什么不买正版呢？有人会问。的确，也买了很多书，但一个是携带不便，另外还是觉得用iPad配合Apple Pencil + Notability记笔记简直太舒服了。所以就开始了漫长的找资源之旅。
#### 目前收集了前端领域的大部分必读书，以及计算机理论基础相关最经典的一些大部头，经过验证基本都是高清完整的。
